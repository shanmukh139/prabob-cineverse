# 🎬 PraBob Cineverse

PraBob Cineverse is a movie information management system developed as an academic mini project using Python, Flask, and MySQL. The application allows users to browse movie details and submit reviews through a simple web interface.

---

## 🚀 Features

- View movie information
- Browse movie details
- Add movie reviews
- Store data using MySQL
- Simple and user-friendly interface

---

## 🛠️ Technologies Used

- Python
- Flask
- MySQL
- HTML
- CSS

---

```
PraBobCineverse/

│── static/

```

---

## ▶️ How to Run

1. Install Python
2. Install Flask

```bash
pip install flask
```

3. Configure the MySQL database.
4. Run:

```bash
python app.py
```

5. Open your browser and visit:

```
http://127.0.0.1:5000/
```
